# UI切图命名规范
## 登陆（login）
```
login_bg  登陆背景图片
login_logo  登陆logo
login_input_n  输入框
login_input_s  输入框选中状态
login_btn_n  登录按钮
login_btn_s  登录按钮选中状态
```
